游戏介绍

休闲娱乐，二次元公平策略竞技，突破常规竞技风格，十分钟就能畅玩一把多人竞技！发育阶段获得的奖励可提升自己的技能/道具等级以及进化路线！竞技过程中还会掉落随机buff，让每场战斗风格都不一样！

背景

余烬之星

在宇宙的深处，漂浮着一颗充满神秘能量的星球——余烬之星。这里的居民世代和平，依赖星球自然能源过着安定且快乐的生活。然而，平静被一场突如其来的天灾打破了。 一颗燃烧着蓝色光芒的陨石从天而降，撞向余烬之星的核心。陨石携带着一种未知的 外星病毒，迅速感染了星球上的动植物。被感染的生物发生了剧烈的变异，成为了极具攻击性的怪物，失去了理智，却拥有了更强大的力量和体内闪烁的 “异能晶核”。 然而，这场灾难并非只有毁灭，也带来了新的希望。研究发现，这些 异能晶核 蕴藏着超乎想象的新型能源，甚至能让余烬之星的文明迎来新的飞跃。然而，采集晶核的过程极其危险，普通士兵根本无法匹敌那些变异怪物。 面对星球存亡的危机，统治者提出了一个疯狂的计划：组织一场竞技试炼，从民间选拔出最强的英雄组成精英部队，对抗变异生物，争取夺回被感染的土地！

病毒来源

病毒来自陨石，实际上是某种外星文明遗留下的生物武器。变异生物的进化似乎有某种规律，随着时间推移，它们的智力和力量都在不断增强。

竞技场的目的

选拔最强战士，获胜者将被授予最高的荣誉，并加入“晶核远征队”，进入感染区域展开最终的清理行动。

游戏玩法

在余烬世界中，竞技分为三个阶段：发育阶段、准备阶段和战斗阶段。以下是每个阶段的详细说明：

发育阶段 (0-4分钟)

玩家在发育阶段可以通过击杀感染的变异者、获取异能晶核等方式来获得技能和道具的升级。这个阶段是为玩家提供资源和成长的时机。

准备阶段 (1分钟)

在准备阶段，玩家将使用在发育阶段获得的奖励来分配自己的技能点，选择buff，并与队友进行沟通，为即将到来的战斗做准备。

战斗阶段 (5分钟)

进入战斗阶段后，玩家将使用自己的技能、道具和地形进行战斗，目标是击败敌方英雄。每位英雄有两条命，若在规定时间内双方英雄没有全部阵亡，将根据击杀数来决定胜负，若击杀数相同，则按当前血量高低决定。

高段位玩家可以选择禁用特定的地图或英雄。

随机元素

战斗阶段中，游戏将随机掉落buff、资源和奖励，使每场战斗的风格都独一无二，增加了战斗的可变性和策略性。

地图特色

余烬中的竞技场为2d地图，地图中的所有地形/平台都可以被毁坏造成坑洞效果，若掉出地图则视为死亡

英雄介绍

冷月（刺客）

特点：极致爆发与灵活

  • 被动：对敌人造成伤害会累计月能量，满时可选择进入满月状态，此状态下提升20%暴击率，移动速度以及伤害加成，并根据伤害量的15%回复自身血量，持续8s； 击杀/助攻分别可使该技能延长2s /1s。
  • 技能1：向指定方向位移，标记范围内血量最低的英雄3s并减速25%； 被标记的目标会减少对自己10%的伤害； 满月状态下对标记的敌人提升25%攻速。 CD 5s

    • Lv2: 减速40% 伤害减少15% 攻速提升40% CD 4s

    Lv3: 减速60% 伤害减少20% 攻速提升60% 且对该目标造成伤害获得的月能量翻倍 CD 3s

  • 技能2： 获得2s 50/75/100%的爆发衰减加速效果，对周围敌人快速旋转2次，造成固定伤害（可暴击）。在满月状态下额外提升20%暴击伤害 CD 5s

    • Lv2: 加速75% 爆伤提升30% CD 4s

    Lv3: 加速100% 爆伤提升40% CD 3s

  • 技能3：冷月高高跃起遁入月中，清空负面状态，无法选中，并获得50/75/100%月能量， 溢出的每1%月能量将提升下一次满月1%被动效果（不提升持续时间）。随后瞬移到地图任意位置（被动：获得15%暴击率提升）。CD 60s

    • Lv2: 获得能量75% 暴击提升25% CD 55s

    Lv3: 获得能量100% 暴击提升40% CD 50s

寒铁战魂（战士）

特点：能抗能打，兼容输出与生存

  • 被动：在冰矛和战斧之间切换状态，冰矛状态下提升20%伤害减免及韧性（减少控制/负面时长）；战斧状态下提升15%伤害加成及全能吸血。
  • 冰矛一：获得一个吸收10%最大生命值的冰晶护盾，持续3s；若被打破或自然消失则对一定范围内的敌人造成初始护盾值100%的伤害； 自然消失时还会将当前护盾值的30%回复自身血量。cd 10s

    • Lv2: 吸收15% 护盾值伤害125% CD 8s

    Lv3: 吸收20% 护盾值伤害150% CD 6s

  • 战斧一：点击蓄力并获得霸体状态，最多2s。再次点击会对正前方扇形区域内敌人造成固定值伤害（1.0倍率）， 蓄力时长与伤害和范围成正比，期间自身移动速度降低50%。 Cd 10s

    • Lv2: 伤害倍率1.2 移速降低35% CD 8s

    Lv3: 伤害倍率1.45 移速降低20% CD 6s

  • 冰矛二：冻一名敌人1.25s，解冻后敌人附带冰冷效果2s，降低20%移速攻速。CD 10s

    • Lv2: 冰冻1.5s 冰冷3s 移速降低30% CD 8s

    Lv3: 冰冻1.75s 冰冷4s 移速降低40% CD 6s

  • 战斧二：向前突进，并使得下三次普攻附带流血效果，每0.5秒损失2%已损失生命值。持续3s，最多叠加3层。Cd 10s

    • Lv2: 损失生命3% 持续4s CD 8s

    Lv3: 损失生命4% 持续5s CD 6s

  • 技能3：切换形态为冰矛/战斧。冰矛状态下造成的流血伤害降低50%，战斧状态下造成的炸盾伤害降低100%。（切换形态不会清空技能冷却） Cd 8s

    • Lv2: 流血伤害降低40% 炸盾伤害降低80% CD 7s

    Lv3: 流血伤害降低30% 炸盾伤害降低50% CD 6s

落羽（游侠）

特点：拥有灵活的远程输出能力，但自身较脆

  • 被动：每2秒获得1圣羽储备，最多12。每次普攻命中会使这个储备冷却加快0.5s。 每个圣羽可以提升自身1%移速，2%攻速。初始为6圣羽。
  • 一：消耗6圣羽，自身获得30%移速加成，使得普攻获得穿透效果持续3s。 若普攻只命中一名敌人，则额外造成20%伤害，若命中多名敌人， 后续的穿透伤害只造成50%伤害。Cd：无冷却时间

    • Lv2: 消耗5圣羽 移速加成40% 额外伤害25%

    Lv3: 消耗4圣羽 移速加成50% 额外伤害30%

  • 二：清除自身携带的所有负面效果，随后在自身范围内降落圣羽攻击， 造成当前圣羽数的50%*固定值的伤害。 若成功解除控制效果还会获得2圣羽储备。Cd 20s

    • Lv2: 圣羽数60% 获得储备4 cd 18s

    Lv3: 圣羽数70% 获得储备6 cd 15s

  • 三：强化自身，被动中普攻命中减少冷却减少为1s。储备上限提升至15， 持续8s。结束时超过12上限的每个圣羽将减少1s此技能冷却。Cd 50s

    • Lv2: 上限提升18 cd 45s

    Lv3: 上限提升22 cd 40s

娜迦（牧师）

特点：辅助队友，拥有强大的保护治疗等增益能力

  • 被动：甘露之雨可以带来增益效果，被娜迦附加护盾或治疗的英雄将获得20%免伤和固定值格挡效果，持续5s。
  • 技能1：若对友方释放，降下甘露之雨治疗范围内血量最低的英雄，使其回复10%最大生命值+固定值的生命治疗效果， 溢出的治疗量会转化为护盾（不可叠加）。若对敌方释放，造成同等治疗量的伤害。Cd 10s

    • Lv2: 回复生命15% cd 8s

    Lv3: 回复生命20% cd 6s

  • 技能2：对范围内所有友方英雄附加自身最大生命值10%的反伤护盾（可叠加）， 护盾被打破前将反弹吸收的50%伤害。Cd 15s

    • Lv2: 生命12% 反弹75% cd 13s

    Lv3: 生命15% 反弹100% cd 10s

  • 技能3：娜迦选择复活一名阵亡的英雄，但此英雄每秒会损失15%最大生命值。 若没有阵亡英雄，则使其死亡后立即复活，但生命值损失会增加至每秒20%。 (该技能复活后的队友死亡不计入死亡次数) Cd 70s

    • Lv2: 生命损失13% 增加至17% cd 65s

    Lv3: 生命损失10% 增加至15% cd 60s

道具介绍

血瓶

恢复生命值。

  • Lv1: 恢复10%+固定值生命，CD45s
    Lv2: 恢复15%
    Lv3：恢复20%
    Lv4：恢复30%
    Lv5: 恢复30%，CD30s

胶水

限制敌方移动跳跃。

  • Lv1: 持续2s，CD30s
    Lv2: 持续时间增加20%
    Lv3：持续时间增加50%
    Lv4：持续时间增加100%
    Lv5：期间敌人受到伤害+20%

蜜蜂出击

投掷蜂窝

  • Lv1: 蜜蜂会对周围所有单位造成2%最大生命值+固定值（包括友方） cd40s
    Lv2: 蜜蜂数量+5
    Lv3：蜜蜂数量+10
    Lv4：造成3%最大生命值
    Lv5：数量x1.5，附带20%减速效果

格挡盾

降低受到的伤害

  • Lv1: 开启后自身受到的伤害降低15%+固定值 持续5s cd60s
    Lv2: 降低20%
    Lv3：持续+2s
    Lv4：单次伤害超过生命上限10%时，额外减伤50%，只触发一次
    Lv5：lv4可触发2次

激光射线

对直线范围内敌人造成伤害

  • Lv1: 对指定直线范围内造成10%最大生命伤害+固定值 cd60s
    Lv2: 范围增加10%
    Lv3：15%最大生命值
    Lv4：伤害免疫减伤/格挡类效果
    Lv5：致盲敌人1s（白屏）

免坑

免疫坑洞效果

  • Lv1: 免疫坑洞效果持续5s
    Lv2: 期间减伤10%
    Lv3：持续+2s
    Lv4：每受到一次伤害，减少0.5s cd
    Lv5：cd45s

核弹

大范围伤害并造成坑洞效果

  • Lv1: 对大范围内区域所有目标（包括友方）造成20%最大生命值+固定值伤害， 巨大坑洞效果，一场战斗只可使用一次。
    Lv2: 范围增加10%
    Lv3：范围增加20%
    Lv4：对免坑的敌人伤害翻倍
    Lv5：目标还将每秒受到1%最大生命值伤害辐射效果，直至死亡。
